# Meeting Protocols

This directory contains all meeting notes and progress documentation for the NSIP Team 2 project.

## Naming Convention

Protocol files follow this format: `YYYYMMDD_Meeting-XX.md`

- **YYYYMMDD:** Date of the meeting
- **XX:** Sequential meeting number (01, 02, 03, ...)
